1. **[Onboarding a New Studio](workflows/onboarding-new-studio.md)** — Get your studio configured
2. **[Onboarding a Teacher](workflows/onboarding-a-teacher.md)** — Add your teaching team

5. **[Onboarding a Member](workflows/onboarding-a-member.md)** — Learn the student experience
6. **[Data Import & Migration](workflows/data-import-and-migration.md)** — Bring in existing data (if applicable)

After that, explore the growth-focused guides as your studio settles in:

7. **[Promos & Discounts](workflows/promos-and-discounts.md)** — Drive initial signups

## App Navigation Reference

Quick reference for where to find things in the Studio Management dashboard:

| Sidebar Item | Path | What's There |
|-------------|------|-------------|
| Dashboard | `/manage` | Today's overview, metrics, alerts |
| Schedule | `/manage/schedule` | Weekly view, sub requests, cancellations |
| Events | `/manage/events` | Workshops, trainings, retreats, series |
| Students | `/manage/students` | Student list, profiles, memberships |
| Teachers | `/manage/teachers` | Teacher profiles, pay rates, availability |
| Offerings | `/manage/offerings` | Class types, descriptions, capacity |
| Promo Codes | `/manage/promo-codes` | Discount codes, usage tracking |
| Financials | `/manage/financials` | Membership types, class packs, transactions |
| Landing Pages | `/manage/landing-pages` | SEO pages, conversion tracking |
| Reports | `/manage/reports` | Attendance, revenue, teacher performance |
| Import Data | `/manage/import` | CSV import with column mapping |
| Settings | `/manage/settings` | Studio info, policies, branding, integrations |

Additional pages (not in sidebar):
- **Onboarding Wizard** — `/manage/onboarding` — Guided first-time setup
- **Member Detail** — `/manage/members/:id` — Individual student profile and history
